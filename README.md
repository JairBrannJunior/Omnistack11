## 💻 Projeto

O BeTheHero é um projeto que visa conectar pessoas que desejam fazer contribuições monetárias a ONG's (Organizações não governamentais) que precisam de ajuda. Foi feito durante a semana Omnistack 11, um evento organizado pela [RocketSeat](https://rocketseat.com.br/), como forma de aprendizado das tecnologias usadas, uma vez que já as conhecia, porém ainda não tinha feito nada na prática. A experiência foi ótima e com certeza despertou ainda mais meu interesse sobre as linguagens.
